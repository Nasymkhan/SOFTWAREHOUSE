PROBLEM: Can't upload profile picture
SOLUTION:
  1. Create /uploads/profile_pics/ folder
  2. Set permissions: chmod 755 uploads/
  3. Check file size (max 5MB)
  4. Check file type (JPG, PNG, GIF only)
  5. Verify PHP upload_max_filesize setting

PROBLEM: Login always fails even with correct password
SOLUTION:
  1. Check database has user record
  2. Verify password hashing works: hash = password_hash($pwd, PASSWORD_BCRYPT)
  3. Check for extra whitespace in credentials
  4. Verify account not suspended

PROBLEM: Session expires immediately
SOLUTION:
  1. Check database datetime is correct
  2. Verify user_sessions table has data
  3. Check token stored in localStorage
  4. Clear browser cache and try again

PROBLEM: "Account suspended" after failed login
SOLUTION:
  Database command to reset:
  UPDATE users SET status='active', login_attempts=0 
  WHERE username='your_username';
