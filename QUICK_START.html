üîß Troubleshooting

Forms not submitting?

1. Check browser console (F12)
2. Verify db.php database credentials
3. Check MySQL is running
4. Check PHP error logs

Admin login not working?

1. Verify admin user exists in database
2. Check username/password are correct
3. Clear browser cookies
4. Check PHP sessions are enabled

Website looks broken?

1. Clear browser cache (Ctrl+Shift+Delete)
2. Check style.css is in same directory
3. Verify file permissions (644)
4. Check browser console for errors

Database connection error?

1. Check database exists
2. Verify credentials in db.php
3. MySQL user has correct privileges
4. Check MySQL is running
